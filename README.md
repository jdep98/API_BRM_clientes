# API_BRM_clientes
﻿# Correr el proyecto

$ cd <API_BRM_clientes>

﻿# Correr la BD

﻿# Requisito, instalar JSON - SERVER desde la terminal

$ npm install json-server
$ cd <API_BRM_clientes>
$ json-server --watch db.json --port 4000

﻿# localhost:3000/clientes
